Intro

Splice is an JavaScript array method, which provides an easy way to modify the contents of an array from any designated index.

Illustration

Consider you had a line of items and you wanted to remove the item located at the end of that line and replace it with a new item. This is a perfect opportunity to use splice

Possible Uses

Splice can be used to add, remove, or replace the content of the targeted array's index. This method changes the original array. Also you could store the removed contents of a splice in a variable.

Using Splice
let exampleArray = [1,2,3];

exampleArray.splice();

First we designate the array we wish to splice, in this case "exampleArray". We then proceed to call the "splice" method on the selected array.

Parameters
exampleArray.splice(2, 1, "item");

The splice method accepts the follow parameters:

  1. start index

    2. The index at which to make changes to the array.

  2. deleteCount (optional)

    3. The number of items from the start index to the number specified to be removed from the array.

  3. item1, item2, item3, ... (optional)

    4. Items added to the array from the start index.

Example
let serenity = [
  "Jack Sparrow",
  "Zoe",
  "Wash",
  "Inara",
  "Jayne",
  "Kaylee",
  "Simon",
  "River",
  "Book",
];

Let's add the proper captain back to the Serenity!

serenity.splice(0, 1, "Mal");

A quick console log shows Serenity has it's captain back.
