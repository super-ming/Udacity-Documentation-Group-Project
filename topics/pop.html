Intro

The pop() method is used for removing the last element at the end of an array and tells you the element that was removed.

Illustration

You can think of an array as a pill box container with a number of detachable sections for each day (Monday - Sunday). Within each section contains a pill to be taken on that day.

Today is Sunday so you want to take out the pill from the very last section of the pill box. 

	let box = ['Monday','Tuesday','Wednesday','Thursday','Friday','Saturday','Sunday'];
	//an array with seven elements for each day of the week
	box.pop();
	'Sunday'
	//After using the pop() method, the removed element is displayed.

Now the box array only contains the following:

	console.log(box);
['Monday','Tuesday','Wednesday','Thursday','Friday','Saturday'];
The Call

In the illustration above, we began by assigning the box variable with an array containing seven string elements, separated by commas. In order to remove the very last element from the array, you call the pop method by adding .pop() after the array name.

	array_name.pop();

After the action is completed, the console automatically returns the element that was removed so you can confirm that the correct element was removed.

Examples

You are hosting a party and you have an array to keep track of who is attending. Your friend Mike just told you that he will be attending so you added him to the array.

	let attendee = ['Sarah','Bill','Tyler','Judy','Mike'];
	//It's a growing list!

Five minutes later, he flaked out and said he will no longer be attending. Now you have to take him out of the list. You can use the pop() method since he is the last person on the list. 

	let flake = attendee.pop();
	console.log(attendee);
	['Sarah','Bill','Tyler','Judy']
	console.log(flake);
	'Mike'
Possible Uses

You can use the pop() method when you need to remove the very last element within an array.

  • Undoing the most recent addition to an array
  • LIFO (Last In, First Out) Accounting
